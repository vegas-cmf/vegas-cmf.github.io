Getting started

In the following article you will know how to build the project on Vegas Library.

Instructions

Install php (at least 5.4 version). Vegas library is based on PHP language. 

//Ubuntu
sudo apt-get install php5
sudo apt-get install php-pear

Install one HTTP server. You can choose Apache or Nginx

Apache Nginx 
//Ubuntu (Apache)
sudo apt-get install apache2

//Ubuntu (Nginx)
sudo apt-get install nginx

Install git

//Ubuntu
sudo apt-get install git

Install phing

//Ubuntu
pear channel-discover pear.phing.info
pear install phing/phing

Install Phalcon

Install Phalcon Library

//Ubuntu
sudo apt-get install php5-dev libpcre3-dev gcc make php5-mysql

Compile Phalcon

cd /var/www
git clone --depth=1 git://github.com/phalcon/cphalcon.git
cd cphalcon/build
sudo ./install

Add extension of phalcon to the php configuration *(In different systems there might be different locations of the php configuration file)

echo 'extension=phalcon.so' > /etc/php5/apache2/conf.d/20-phalcon.ini

Create your project using vegaser tool

cd /var/www
mkdir vegas-test
cd vegas-test
wget https://bitbucket.org/amsdard/vegaser/downloads/vegaser.phar --no-cache
sudo php vegaser.phar build-project
