Getting started

In the following article we show how to prepare environment for projects based on Vegas CMF.

Requirements

Vegas CMF need at least PHP in version 5.4. Make sure that it's already installed in your system:
php -v

Vegas CMF based on PhalconPHP Framework implemented as C extension.
Before you start building project, check the following link to install extension.
To ensure that extension is correctly installed, enter the following command in the terminal:
php -m | grep phalcon

As a main database we usually use MongoDB. To install it in your system follow this link

Vegaser

To make building a project as simple as possible, we prepared a small tool for generating a sample project based on Vegas CMF.
